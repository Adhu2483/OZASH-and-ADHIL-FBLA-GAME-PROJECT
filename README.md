# Project Overview

**Time Traveler: Apollo Mission** is a browser-based educational game built using **HTML, CSS, and JavaScript**.

In the game, the player becomes a **time traveler sent to the 1960s** to help NASA complete the historic **Apollo Moon Mission**. Players must collect rocket components, learn historical facts, and complete multiple levels before time runs out.

The game combines **interactive gameplay** with **historical learning**, teaching players about the Space Race and the Apollo 11 Moon Landing in 1969.

---

# Features

* Player movement using **arrow key controls**
* **Score system** based on collected items
* **Two levels** of gameplay
* **Timer-based challenge**
* **Win and lose conditions**
* Randomly displayed **historical facts**
* Simple and responsive **browser-based gameplay**

---

# Technologies Used

* **HTML** – Game layout and canvas container
* **CSS** – Styling and visual presentation
* **JavaScript** – Game logic, player movement, scoring, and level management
* **HTML5 Canvas** – Rendering the game graphics

# How to Run the Game

1. Download or clone the project folder.
2. Open the folder on your computer.
3. Double-click **index.html**.
4. The game will launch automatically in your web browser.

No additional software or installation is required.

---

# How to Play

* Use the **Arrow Keys** to move the player.
* Collect rocket parts to earn **points**.
* Each collected item increases your score.
* Reach the required score to progress to the **next level**.
* Complete both levels before the **timer reaches zero**.

If the timer runs out, the mission fails.

---

# Educational Content

This game teaches players about important events from the **1960s Space Race**, including:

* Apollo 11 Moon Landing (1969)
* Neil Armstrong becoming the first person on the Moon
* The role of NASA during the Space Race
* Key historical milestones of lunar exploration

Historical facts appear throughout gameplay to help reinforce learning.
